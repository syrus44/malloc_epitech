malloc_epitech
==============

Malloc : projet Epitech. Ce projet est une refonte de la commande malloc présente sur les systèmes Unix. Elle permet une
allocation statique ou dynamique de la mémoire (RAM). Avec malloc, on peut tout faire :D

Langages : C
